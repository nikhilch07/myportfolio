## Project Structure
The project is organized into the following main directories:
- `src/`: Contains the source code for the portfolio website.
  - `components/`: Reusable React components.
  - `pages/`: Different pages of the portfolio (e.g., Home, About, Projects).
  - `assets/`: Images, stylesheets, and other static assets.
- `public/`: Publicly accessible files.
- `README.md`: This documentation file.
- `package.json`: Project configuration and dependencies.

## How to Navigate
To explore the portfolio, follow these steps:

1. Clone the repository to your local machine using `git clone https://github.com/nikhilch07/myportfolio.git`.
2. Navigate to the project directory: `cd portfolioject`.
3. Install the necessary dependencies using `npm install` or `yarn install`.
4. Start the development server with `npm run dev` or `yarn run dev`.
5. Open your web browser and go to `http://localhost:5173` to view the portfolio.
